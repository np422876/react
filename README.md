Customer Responsive Management for Real Estate

Project Description:
A full-stack Real Estate Management System built using React, Node.js, Express, MongoDB, and JWT Authentication. The application allows users to browse properties, add new properties, edit property details, delete properties, save favorite properties, and manage listings through a responsive user interface.

   - Authentication

      2.User Login
      3.JWT Authentication
      4.Protected Routes

   - Property Management

      4.View Property Details

* Installation

  - Clone Repository
     git clone https://github.com/np422876/react.git
       
  - Frontend Setup
     cd client
     npm install
     npm run dev

  - Backend Setup
     cd server
     npm install
     npm start
